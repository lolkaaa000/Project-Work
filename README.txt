Машинное обучение: Framingham heart disease

Проект: Анализ ишемического заболевания сердца

Выполнили
- Амандос Анель  
- Ернияз Арман

Цель проекта
Проект направлен на построение моделей машинного обучения для предсказания 10-летнего риска ишемической болезни сердца (CHD) на основе медицинских данных пациентов.  
Используются:
- Линейная регрессия (с нуля)  
- Логистическая регрессия (с нуля)  
- Decision Tree (sklearn)  

Проект включает эксперименты с гиперпараметрами, визуализацию, интерактивные виджеты и сравнительный анализ моделей.

Данные
- Источник: [Kaggle Framingham-heart-disease](https://www.kaggle.com/datasets/ppranayk/framingham-heart-disease)  
- Размер: 4238 строк, 16 столбцов  
- Описание признаков:

| Признак          | Описание                                                                 |
|------------------|--------------------------------------------------------------------------|
| male             | Пол (1 = муж, 0 = жен)                                                   |
| age              | Возраст                                                                  |
| education        | Образование                                                              |
| currentSmoker    | Курит ли в настоящий момент (1 = да, 0 = нет)                            |
| cigsPerDay       | Количество сигарет в день                                                |
| BPMeds           | Принимает ли лекарства от давления (1 = да, 0 = нет)                     |
| prevalentStroke  | Была ли ранее инсульт (1 = да, 0 = нет)                                  |
| prevalentHyp     | Наличие гипертонии (1 = да, 0 = нет)                                     |
| diabetes         | Диабет (1 = да, 0 = нет)                                                 |
| totChol          | Общий холестерин                                                         |
| sysBP            | Систолическое давление                                                   |
| diaBP            | Диастолическое давление                                                  |
| BMI              | Индекс массы тела                                                        | 
| heartRate        | Пульс                                                                    |
| glucose          | Уровень глюкозы                                                          |
| TenYearCHD       | Целевая переменная: развитие CHD за 10 лет (1 = да, 0 = нет)             |

Предобработка данных
1. Заполнение пропущенных значений:
   - числовые → среднее  
   - категориальные → мода  
2. Нормализация числовых признаков (StandardScaler)  
3. Разделение на обучающую (80%) и тестовую (20%) выборки  

Реализация моделей
1. Линейная регрессия
- Реализована с нуля на NumPy  
- Градиентный спуск: `learning rate = 0.01`, `epochs = 1000`  
- Визуализация scatter plot и линии регрессии с доверительным интервалом  

2. Логистическая регрессия
- Реализована с нуля на NumPy  
- Функция: `sigmoid`  
- Градиентный спуск: `learning rate = 0.1`, `epochs = 1000`  
- Метрики: Accuracy, Precision, Recall, F1-score, ROC AUC, Confusion Matrix  
- График Log Loss  

3. Decision Tree
- Модель: `DecisionTreeClassifier(max_depth=5)` (sklearn)  
- Метрики: Accuracy, Precision, Recall, F1-score, ROC AUC, Confusion Matrix  
- Визуализация дерева решений  

Эксперименты с гиперпараметрами
- Влияние learning rate и количества эпох на сходимость градиентного спуска  
- Сравнение логистической регрессии и Decision Tree по метрикам  
- Анализ: Decision Tree улавливает нелинейные зависимости, но может переобучаться; логистическая регрессия стабильна и интерпретируема
